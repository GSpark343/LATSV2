Hardware Utilizado

Componentes del sistema de adquisición de datos

Microcontrolador Arduino Nano

El Arduino Nano es el cerebro del sistema. Este microprocesador de bajo costo y tamaño compacto se encarga de leer los datos de los sensores de temperatura y humedad, procesarlos y almacenarlos en la tarjeta MicroSD. Su arquitectura basada en el ATmega328P ofrece un equilibrio ideal entre rendimiento y consumo energético para aplicaciones de monitoreo en tiempo real.


Especificaciones clave:

Voltaje: 5v / 3.3v, Atmega328P, arquitectura: AVR de 8 Bits, Frecuencia: 16 MHz, Memoria flash: 32 KB, Pines digitales: 14, Entradas analógicas: 8, Comunicación: I2C, Tamaño: 45mm x 18mm

✦

Sensor DHT22

El DHT22 es un sensor digital de temperatura y humedad relativa de alta precisión. Utiliza un protocolo de comunicación de un solo cable, lo que simplifica el cableado. Es ideal para entornos de bodega donde se requiere monitoreo estable en rangos de temperatura entre -40°C y 80°C y humedad del 0% al 100% RH.


Especificaciones clave:

Temp: ±0.5°C · Humedad: ±2–5% RH · Protocolo: Single-wire · Voltaje: 3.3–5V

✦

Módulo MicroSD

Para el almacenamiento local de datos, se usa un módulo lector de tarjeta MicroSD conectado al Arduino Nano. Los datos de temperatura y humedad se escriben en formato CSV con marca de tiempo, permitiendo exportación directa para análisis posterior en esta plataforma web.


Especificaciones clave:

Interfaz: SPI · Capacidad: hasta 32GB · Formato: FAT32 · Voltaje: 3.3V

Memoria SD

Para el almacenamiento local de datos. Los datos de temperatura y humedad se escriben en formato CSV con marca de tiempo, permitiendo exportación directa para análisis posterior en esta plataforma web.


Especificaciones clave:

Interfaz: SPI · Capacidad: hasta 32GB · Formato: FAT32 · Voltaje: 3.3V

Módulo RTC DS3231

Un módulo RTC (Real Time Clock) DS3231 es un reloj de tiempo real preciso que mantiene la fecha y la hora incluso cuando el Arduino no está alimentado gracias a su batería interna. Se comunica con el Arduino por I²C para obtener el tiempo actual (hora, minutos, segundos, fecha) y así poder marcar cada medición con una marca temporal real en vez de depender del temporizador interno del microcontrolador, que se reinicia con cada reset o apagado del sistema.


Especificaciones clave:

Interfaz: SPI · Capacidad: hasta 32GB · Formato: FAT32 · Voltaje: 3.3V
