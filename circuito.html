Esquema del Sistema

Diseño eléctrico del circuito de sensado

El circuito fue diseñado para operar de manera autónoma dentro de la bodega, registrando temperatura y humedad en intervalos regulares. La alimentación se realiza mediante batería LiPo de 3.7V con módulo de carga solar opcional para instalaciones sin toma eléctrica cercana.

Tabla de Conexiones

Las conexiones entre el Arduino Nano y los módulos se realizan con resistencia pull-up de 10kΩ en la línea de datos del DHT22. El módulo MicroSD se conecta por bus SPI y el RTC DS3231 mediante protocolo I2C.

Arduino Nano + DHT22

Sensor de temperatura y humedad. Se conecta al Arduino Nano en la línea de datos D3.

VCC → 5V

GND → GND

DATA → D3

Arduino Nano + Módulo MicroSD

Módulo de almacenamiento conectado mediante bus SPI. Permite guardar los registros de temperatura y humedad en una tarjeta microSD.

CS → D4

SCK → D13

MISO → D12

MOSI → D11

5V o VCC → 5V

GND → GND

Arduino Nano + RTC DS3231

Módulo de reloj en tiempo real con comunicación I2C. Proporciona la marca de tiempo exacta para cada registro almacenado.

VCC → 5V

GND → GND

SDA → A4

SCL → A5

Implementación Física

El prototipo fue implementado en protoboard para validación inicial. La versión final se encapsula en una caja ABS IP65 para protección contra humedad, condición fundamental para su operación dentro de la bodega donde la humedad relativa supera regularmente el 70%.
